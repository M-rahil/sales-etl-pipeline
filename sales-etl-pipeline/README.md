#### \# Sales Data ETL Pipeline

#### 

#### \## Overview

#### This project implements a simple ETL (Extract, Transform, Load) pipeline for processing sales data.

#### 

#### \## Process

#### \- Extract: Load raw sales data from a CSV file

#### \- Transform: Clean data and create new features using Python (Pandas)

#### \- Load: Store processed data into a new file

#### 

#### \## Technologies Used

#### \- Python

#### \- Pandas

#### \- SQL (basic concepts)

#### 

#### \## Features

#### \- Data cleaning (handling missing values)

#### \- Revenue calculation (quantity × price)

#### \- Automated data processing pipeline

#### 

#### \## How to Run

#### 1\. Install dependencies:

#### &#x20;  pip install pandas

#### 

#### 2\. Run the script:

#### &#x20;  python etl.py
